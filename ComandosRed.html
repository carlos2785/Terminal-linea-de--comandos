  • Comandos de red
    1. ifconfig

      2. Permite listar los puertos de red y su respectiva configuración

    2. ping

      3. Permite establecer si hay o no conexión con un servidor o página web

    3. wget

      4. wget www.google.com= descarga un archivo con extensión .html y lo guarda en el directorio

    4. traceroute

      5. traceroute www.google.com= muestra la ruta de equipos que sigue para estabelcer la conexión con el servidor

    5. netstat

      6. netstat -i= muestra una tabla con las conexiones activas en el equipo
