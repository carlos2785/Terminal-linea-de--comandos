La Terminal y Línea de Comandos

  • La Terminal

    • Es la ventana que nos muetsra la Shell

  • Linea de Comandos

    • Es un programa que ejecuta comandos directamente en el sistema operativo

  • Comandos de la terminal
    1. pwd

      2. Es la ventana que nos muetsra la Shell

    2. cd

      3. Permite ingresar a las carpetas

    3. cd..

      4. Permite devolverse entre las carpetas

    4. ls

      5. Lista los archivos de un directorio, ls -lh:los muestra más legibles y ls -a: muestra los archivos ocultos

    5. file

      6. Junto con el nombre del archivo, permite mostrar características importantes del mismo

    6. tree

      7. Lista los archivos del directorio como un arbol, es decir mostrando carpetas y sub-carpetas, se puede hacer de cualquier directorio

    7. mkdir

      8. Permite crear carpetas en directorios específios

    8. touch

      9. Crea archivos en el directorio especificado

    9. mv nombreArchivo ../carpetaDestino

      10. Permite mover un archivo especificando su nombre a la carpeta o directorio de destino, especificando el nombre de la carpeta de destino y su ruta. También sirve para renombrar los archivos dentro del mismo directorio o para renombrar los directorios

    10. rm

      11. Permite borrar archivos

    11. rm -ri

      12. Permite borrar archivo por archivo el contenido del directorio, incluido el mismo directorio

    12. head nombreArchivo.exptension

      13. Permite leer un archivo, si se agrega -n #, permite leer la # lineas del archivo

    13. less nombreArchivo.exptension

      14. Permite por ejemplo buscar una plabra de la siguiente manera: /inteligencia y el resultado son todas las palbras inteligencia relsaltadas en la linea de comando
