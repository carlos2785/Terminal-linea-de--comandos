  • Comandos de busqueda
    1. printenv

      2. find ./ -name file= este comando muestra todos los archivos cuyo nombre es file
      find ./ -type d -name carpeta1= busca en el directorio que se ubique, todos los directorios(carpetas) que se llamen: carpeta1
      find ./ -size 20M= busca en el directorio que se ubique, todos los archivos con peso mayor a 20MB

    2. grep

      3. grep Towers movies.csv= Busca dentro del archivo movies, la palabra Towers y muestra las coincidencias
      grep the movies.csv= busca en el archivo movies las coincidencias con la palabra "the"
      grep the -i movies.csv= busqueda similar al anterior pero que incluya la mayúscula inicial
      grep -c the movies.csv= cuenta las coincidencias encontradas
      grep -vi towers movies.csv= ignora las coincidencias de la palabra towers tanto en mayúsculas como en minúsculas
      wc movies.csv= cuenta las palabras, letras y peso en bits del archivo
      wc -c movies.csv= si necesito solo el tamaño en bits, este comando lo genera
