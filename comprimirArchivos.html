  • Comprimir archivos
    1. tar

      2. Este comando permite comprimir directorios o archivos
      tar -cvf ToCompress.tar ToCompress= tar -cvf nombreArchivoComprimido nombreDirectorioAComprimir

    2. tarz

      3. Este comando también permite comprimir directorios o archivos, es más recomendado
      tar -cvzf ToCompress.tar.gz ToCompress= tar -cvf nombreArchivoComprimido nombreDirectorioAComprimir

    3. zip

      4. permite comprimir y descomprimir docuemtnos o carpetas en.zip
      zip -r ToCompressInZip.zip ToCompress= descomprime el archivo ToCompress.zip
